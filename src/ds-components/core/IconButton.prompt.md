Icon-only button for toolbars, table rows, and dense controls. Always pass `label` (accessibility + tooltip).

```jsx
<IconButton label="More actions"><MoreIcon /></IconButton>
<IconButton label="Filter" bordered size="md"><FilterIcon /></IconButton>
```

Sizes `sm`/`md`/`lg`. Set `bordered` to give it a card surface + hairline border on empty backgrounds.
