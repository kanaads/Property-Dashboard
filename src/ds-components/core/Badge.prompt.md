Compact status / category label.

```jsx
<Badge tone="success" dot>Active</Badge>
<Badge tone="brand" pill>Multifamily</Badge>
<Badge tone="gold" appearance="solid">Off-market</Badge>
```

Tones: neutral, brand, gold, success, warning, danger, info. Appearances: soft (default), solid, outline. Options: `pill`, `dot`, `icon`.
